Python 基础语法

本部分内容，将从 Python 最基本的语法开始，逐步学习变量、数据类型等内容。网络上有较多的参考视频和网站，可自行寻找参考学习，宜精不宜多。

  1. 基础的基础
    从print开始第一个程序

    print('Hello world')
print('life is short, but i use python')
print('292930')
print('my name is gong')
print('my name is zhang')

    变量
    为了方便存储部分内容，引入变量概念，就像手机号对应的人名一样 注意，python的执行方式是一行一行从前到后执行，如果没有先定义变量，限制性含变量的程序，系统找不到变量会报错

    name = 'Gong'
print(name)
name = 'zhang'
print(name)
# 变量的第一个字符必须以字母（a-z, A-Z）或下划线 _ ，其他的部分由字母、数字和下划线组成
# 区分大小写
age = 20
size = 'large'

'''
2nd_place = "silver"    # 错误：以数字开头
user-name = "Bob"       # 错误：包含连字符
$price = 9.99          # 错误：包含特殊字符
class = "Math"          # 错误：使用关键字
for = "loop"           # 错误：使用关键字
'''

    关键字是系统已经使用的名字，你不能再使用，可用以下命令查看

    import keyword
print(keyword.kwlist)

    注释
    上面的三个引号里的内容为注释，注释用于提供文字说明，更好理解程序,包括#内容和'''内容'''两种方式
    采用#开头的，后面都是注释，注释不执行，可在程序前，也可在程序后
    采用'''内容'''进行注释，可以是单引号，也可以是双引号，新版本似乎"""有所变化，建议采用'''

    # 这里是注释
# 这里还是注释

'''
这里是一大段注释
通常用于大程序的前期描述
或者独立程序的描述
'''

"""
与单引号
注释一致
"""

    换行与缩进
    python最具特色的就是使用缩进来表示代码块，通常为一个tab，缩进表示这一块内容从前到后运行，如下所示：

    if True:
    print ("True")
    print ('缩进很重要！')
else:
    print ("False")
  print ("False")    # 缩进不一致，会导致运行错误

    当程序内容过多，一行写不完时，可以换行，如下所示

    item_one = 1
item_two = 2
item_three = 3

total = item_one + \
    item_two + \
    item_three
print(total)

    当输出需要换行时，可以用\n(换行),\t(制表符，1个tab=4个空格)等字符控制来执行

    print('happy new day!\t')
print('happy new everyday!\n')
print('another happy day')

    当字符串引号中含引号时，采用双引号包裹单引号方式，或者用转义符\

    s_str = "i'm good"
s_str1 = 'i\'m good'
print(s_str)
print(s_str1)

  2. 基本数据类型
    常见的python基础数据类型包含以下内容：

    • number(数字)，主要操作相互转化和简单运算(+,-,*,/,%,**,//)
    • string(字符串),主要操作有拼接、索引、切分等,字符串常见命令：split,join,strip,replace,lower,upper,len
    • list(列表)，有拼接、索引、切分等,常见命令有copy,in, not in,append,remove,pop
    • tuple(元组)，类似这样结构，相校list，tuple不能修改
    • dict(字典),key==>value(可以任意数据), dic = {key1:value1,key2:value2}
    • set

    2.1 数字

    数字类型包括整数、浮点数、复数等，如下所示:

    a = 5       # 整数 int
b = 3.14    # 浮点数 float
c = 2 + 3j  # 复数 complex

    两种不同的数字类型

    x = 8
print(x)
x = 8_000_000
print(x)

    两种不同的浮点数类型

    y = 8.52452
print(y)
y = 8.2e-3
print(y)

    整型、浮点型、字符串相互转化

    d = 8.5544
int(d)
d = 900
float(d)
str(10)

    当不确定数据类型时，采用type命令验证数据，如下：

    type(c)
type(d)

    数字之间的相互运算

    m = 5
n = 2
x = m - n
print(x)
#类似的
print(5 + 2)   # 加法 7
print(5 - 2)   # 减法 3
print(5 * 2)   # 乘法 10
print(5 / 2)   # 除法 2.5
print(5 % 2)   # 取余 1
print(5 ** 2)  # 幂运算 25
print(5 // 2)  # 整除 2

    控制小数位数，可以用round命令

    x2 = 4
y2 = 6
xy2 = x2/y2
print(xy2)
round(xy2,2)

    2.2 字符串
    字符串常见的使用有索引、切片、拼接、重复以及删除等，实现对字符串的操作

    string1 = 'This is a string'
string2 = 'and'
string3 = '300'

# print(string1)
# print(string2)
# print(string3)

print(string1[0]) # python从0开始索引
print(string1[-2])  # 可从前到后，也可以从后到前进行索引
print(string1[1:4]) # 切片是左边包含右边不包含
print(string1[4:])
print(string1[:-6])

string4 = string1+' '+string2+string3 #拼接
print(string4)
string4 = string1*3 #重复
print(string4)

    除此之外，还有一些常见的操作命令 strip用于删除字符串两端多余内容，常见于删除空格，string.strip(''),引号内为对应查找内容，可为空

    string6 = '   $$$ Remove unwanted chararcters form this string.\t\t       \n+++___   ' 
string6_1 = string6.strip() # 删除左右两边空格
string6_1 = string6.lstrip() # 删除左边空格
string6_1 = string6.rstrip() # 删除右边空格
print(string6_1)
string6_1 = string6.strip(' $_-+\n\t') ###if start or end with $_-+
print(string6_1)

    replace用于删除字符串两端多余内容，常见于删除空格，string.replace('','',2),第一个引号内为对应查找内容，第二个为替换的内容，数字为替换次数，默认为全部替换

    string7 = 'replace the space in this sentence'
string7_1 = string7.replace(' ','_',2)
print(string7_1)

    大小写很关键，python是区分大小写的，所以为了统一，可用lower或upper改成都是小写或者都是大写

    aa = 'Alpher'
AA = 'alpher'
if AA == aa:
    print('True')
print(aa.upper())
print(aa.lower())
if AA.upper() == aa.upper():
    print('TRUE')

    字符串的格式化输出，实现快捷替换。包括%,format,{}三种方式

    #1st way
'Hi, %s, you have $%d.'%(s_str, 10000000) # here %s stand for str，%d stand for int, %f for float
#2nd way
print('Hi,{0},you looks good, you have {1:.2f}.'.format('Gong',10000.22))
print('Hi,{0},you looks good, you have {2:.1f}.'.format('Gong',10000.22,202311))
#3rd way
name = 'Gong'
goods = 1000000
print(f'Hi,{name},you looks good ,and you have {goods:.1f}')

    split用于分割字符串，string.split('',2),第一个引号内为对应查找内容，从找到的内容处开始切分，数字为切分次数，默认为全部切分

    string5 = 'This is a test for split'
string5_1 = string5.split(' ')
print(string5_1)
string5_2 = string5.split(' ',2) #split only 2
print(string5_2)
string6 = 'This,is,a,another,test,for,split'
string6_1 = string6.split(',')
print(string6_1)

# #example
filename = '哥谭市空气质量小时数据(2022年01月-05月)'
s_output = filename.split('(')
print(s_output)
s_output = s_output[1].split(')')
print(s_output)
print(s_output[0])
#another way
file_out = filename[12:-1]
print(file_out)

    2.2 列表
    列表用于存放不同类型的数据，类似一个抽屉，抽屉里的内容有顺序，可修改，也可包含不同类型

    list1 = [1,2,3,4,5]
list2 = ['a','b','c','d','e']
a_list = [12,13,'data','today',['你','真棒'],12]

print(a_list[4]) # index,与string类似
print(a_list[1:3])   # slice,与string类似
print(a_list.index(['你', '真棒'])) #if we don't know the index,return 1st one
print(a_list.index(12))
print(a_list[-2][0]) #可两层叠加

    新建一个空的list，用于存放内容，新增内容可用append

    e_list = []
print(e_list)
e_list.append('one') #添加到后面
e_list.append(17)
e_list.append(['我们','真棒'])
print(e_list)
print(type(e_list))

    复制一个list，在新的list中进行操作，这样原来的list就不会变

    c_list = a_list.copy()
print(c_list)

    list的拼接可以直接相加，也可以用新的list放入原来内容

    d_list = list1+list2
print(d_list)
print([list1,list2])

    修改list中数据，找到对应的数据索引，给这个索引数据赋新的值就行

    a_list[-2] = ['xxx', '真棒']
print(a_list)

    删除数据，可用pop和remove两种方式，两种方式有略微的差异，如下 pop是序号删除，删除的这个值可用，有点像提取

    print(a_list)
re = a_list.pop(-2) #序号删除
print(re)
print(a_list)

    remove是值删除，直接删除对应的值，但只是查找到的第一个

    re = a_list.remove(13) #值删除
print(re)
print(a_list.remove(12)) #delete 1st one
print(a_list)

    数字list可以简单计算

    b_list = [2,3,4,5,6,7,8,5,4,7,10,3,3,2,5,6,11]
print(len(b_list)) #数据多少个
print(max(b_list)) #最大
print(min(b_list)) #最小
print(b_list.count(3)) #计算出现次数

print('b_list has {} elements.'.format(len(b_list)))#len,max,min,count

    2.3 元组数据 元组类似列表，但不可修改，元组使用小括号(),英文的小括号

    tup1 = (1,2,3,4,5)
tup2 = 'a','b','c','d','e'
print(tup1)
print(type(tup1))
print(tup2)
print(tup2[0])
# tup1[0] = 5  #会报错

#delete tuple element,delete all tuple
del tup2
print(tup2)

    2.4 字典数据 字典是另一种可变容器模型，且可存储任意类型对象，key==>value(可以任意数据), 每个对之间用逗号(,)分割，整个字典包括在花括号 {} 中，类似这样的结构：dic = {key1:value1,key2:value2}
    value可以取任何数据类型,既可以是标准的对象，也可以是用户定义的， key必须是不可变的，且不允许同一个键出现两次，所以可以用数字，字符串或元组充当，而用列表就不行 创建时如果同一个键被赋值两次，后一个值会被记住

    dict1 = {'Name':'Gong','number':'1292847',\
        'ID':'365739','hobby':['basketball','reading']}
print(dict1)

e_dict = {} # 使用大括号 {} 来创建空字典

print(len(dict1)) # 查看字典里的数量
print(dict1['Name'])  #访问字典里key为Name的数据
print(dict1.keys()) #查找所有的key
print(dict1.values()) #查找所有的值
print(dict1.items()) #以配对方式查找所有的配对

dict1['Height'] = 171 #新增字典元素
dict1['Name'] = 'SUPER'  #更新NAME
print(dict1)

re = dict1.pop('Height') #删单一的元素
print(dict1)

print(dict1.get('Name'))
