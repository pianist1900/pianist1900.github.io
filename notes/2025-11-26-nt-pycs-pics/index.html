python学习｜数据可视化

Matplotlib

前面我们已经学习了利用Pandas 读取数据，并对数据进行简单的分析，本节我们讲解一下利用Python来做图。Python有多个画图的库，例如Matplotlib、seaborn等，先简单学习Python中最流行的数据可视化库之一：Matplotlib。

1.Matplotlib库安装

首先，确保你的 Python 环境已经安装了 Matplotlib。如果你还没有安装，可以使用 pip 命令轻松搞定： pip install matplotlib，库安装完成后在程序中导入库：

import matplotlib as mpl
import matplotlib.pyplot as plt
import numpy as np

2.画图的基本元素

在 Matplotlib 中，理解 Figure (画布) 和 Axes (坐标系/子图) 这两个核心概念至关重要。

  • Figure (画布)：你可以把它想象成一张白纸或一个窗口。所有的图表都绘制在这张“纸”上。一个 Figure 可以包含一个或多个Axes。
  • Axes (坐标系/子图)：这是你真正绘图的区域。它包含x轴、y轴、刻度、标签和实际的数据点。一个 Figure 可以有多个 Axes，每个Axes 都是一个独立的绘图区域。

其他要素如下图所示； 详细要素介绍可见下表：

3.简单的线图

线图和散点图，需要对应的x,y轴的数据，这里我们简单构造几个点数据,如下所示：

x = [0,1,2,3,4], y = [0,2,4,6,8]

上面数据代表(0,0),(1,2)(3,6),(4,8)几个点，然后我们在程序中进行简单的画图，见下面程序：

#导入依赖库
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
#预设数据
x = [0,1,2,3,4]
y = [0,2,4,6,8]
#画图
plt.figure(figsize=(5,3),facecolor='white') #定义画框
plt.plot(x,y, label='2x', color='red', linewidth=2, marker='s', linestyle='--', markersize=10, markeredgecolor='blue') #画线图

然后我们得到如下的图： 上面的程序中，plt.figure定义一个画布，主要定义画布的大小和颜色，在定义好大小和颜色后，plt.plot(x,y)开始画图折线，其中x,y分别对应x值，y值，即上面列表的两个值，两两对应就成为一个确定的坐标，label是我们这个画图线的名称，方便图例时显示，color是折线图的颜色，linewidth是线的宽度，marker是每个坐标点的方式，linestyle是线图的折线样式，markersize是标记点的大小，markeredgecolor是标记点外框的颜色。 Python画图主要是在画对应图的时候，用不同的参数来细化图的表达。

常见的参数有：

  • 颜色：颜色有几种表达方式，一种是颜色名称如red、black、blue、green等，另一种为颜色名称的缩写，如r、b、u、g等，除了这两种外，颜色还可以用16进制的颜色代码，如#FF5555，常用的颜色简有’b’ (蓝色), ‘g’ (绿色), ‘r’ (红色), ‘c’ (青色), ’m’ (洋红色), ‘y’ (黄色), ‘k’ (黑色), ‘w’ (白色)。
  • 标记，标记有两种方式，一种是标记的中文名称，如star、square，另外一种是简易符号，如*、^、等等，常用的标记有：‘o’ (圆圈), ‘^’ (上三角形), ’s’ (正方形), ‘D’ (菱形), ‘*’ (星形), ‘+’ (加号)。
  • 线类型：线类型也有两种方式，名称和符号等，常用的线型： ‘-’ (实线), ‘–’ (虚线), ‘-.’ (点划线), ‘:’ (点线)。 在Matplotlib官方文档 - plot函数 中有更详细的参数说明。

同时Matlibpolt也支持一种便捷的方式显示，fmt = '[color][marker][line]'，如上面画图程序的我们可以缩写为b^--，在程序中：

plt.plot(x,y, 'b^--', label='2x')

在基本的图形完成后，可以根据Axes的画图要素逐步完善剩下的图像内容，通常有标题、XY轴、图例等 标题。增加图像标题，通过plt.title()函数来完成，见代码

plt.title('Our first graph',fontsize=18,fontname='Times New Roman',pad=10,loc='center')

上面代码中，字符串为画图的标题，后面参数依次为字体大小，标题的字体，距离画框距离，以及对应的位置。
XY轴。接着可以增加xy轴信息，利用plt.xlabel()和plt.ylabel()来完成，可以提前定义字体的一个字典label_fontdict，然后在xy轴中直接调用，如下所示：

label_fontdict={'fontname': 'Comic Sans MS', 'fontsize': 20}
plt.xlabel('x axis',fontdict=label_fontdict)
plt.ylabel('y axis',fontdict=label_fontdict)

XY轴刻度。当然刻度设置也不能少，用plt.xticks和plt.yticks分别调整x、y轴的刻度, 用plt.ylim()和plt.xlim()来调整刻度范围。rotation=-20代表刻度的倾斜度

plt.xticks(x,fontsize=20,rotation=-20)
plt.yticks(size=20)
plt.ylim(0,16)
plt.xlim(0,4)

图例。画图不能少了图例，采用plt.legend()来刻画图例，图例参数中ncol代表图例的列,frameon代表图例的方框,loc代表图例的位置，可以用best来自动选择最优位置。

plt.legend(fontsize=20,ncol=2,frameon=False,loc='best')#loc='upper right'
plt.grid(linestyle='--')

保存。在完成绘图后，可用plt.savefig()来保存图片，参数中dpi为图片清晰度

plt.savefig('our first graph',dpi=300,bbox_inches='tight')
plt.show()

多条线。以上为一条折线，如果要多条线段如何表示呢，别着急，Matplotlib中一个figure可以叠加多个内容，我们可以不断增加plt.plot()来实现，如下面完整的画图代码：

x = [0,1,2,3,4]
y = [0,2,4,6,8]
plt.figure(figsize=(5,3),facecolor='white')

title_fontdict={'fontname': 'Comic Sans MS', 'fontsize': 20}
label_fontdict={'fontname': 'Comic Sans MS', 'fontsize': 20}

plt.plot(x,y, label='2x', color='red', linewidth=2, marker='s', linestyle='--', markersize=10, markeredgecolor='blue')

# # #2nd line
x2 = np.arange(0,4.5,0.5)

# # # Plot part of the graph as line
plt.plot(x2[:6], x2[:6]**2, 'r', label='X^2')
# # # # Plot remainder of graph as a dot
plt.plot(x2[5:], x2[5:]**2, 'r--')

plt.title('Our First Graph!', fontdict={'fontname': 'Comic Sans MS', 'fontsize': 20})
plt.xlabel('x axis',fontdict=label_fontdict)
plt.ylabel('y axis',fontdict=label_fontdict)

# # #ticks
plt.xticks(x,fontsize=20,rotation=-20)
plt.yticks(size=20)
plt.ylim(0,16)
plt.xlim(0,4)
# # #legend
plt.legend(fontsize=20,ncol=2,frameon=False,loc='best')#loc='upper right'
plt.grid(linestyle='--')

plt.savefig('our first graph',dpi=300,bbox_inches='tight')
plt.show()

4.柱状图

类似的，我们可以快速的画柱状图，与线图不同的是，需要柱状图构建的数据不同，它需要类别和对应的两两对应，见代码：

import matplotlib.pyplot as plt
import numpy as np

categories = ['A', 'B', 'C', 'D', 'E']
values = [23, 45, 56, 12, 39]

plt.figure(figsize=(7,5), facecolor = 'white',dpi=100)

plt.bar(categories, values, color=['skyblue', 'lightcoral', 'lightgreen', 'orange', 'purple'])

plt.title("Bar Chart Example")
plt.xlabel("Categories")
plt.ylabel("Values")
plt.show()

执行上述代码会得到如下的图，bar图在给参数的时候，需要先给类别,然后再给对应的值,最后是参数。

如果想要柱状图的堆叠图，那需要构建多组数据，即一组属于底层数据，另外位于上层组的数据，见如下代码：

import matplotlib.pyplot as plt
import numpy as np
labels = ['A', 'B', 'C']
values = [1,4,2]
bottom = [3,1,3]

plt.figure(figsize=(5,3), facecolor = 'white',dpi=100)
bars1 = plt.bar(labels,bottom,width=0.8,color='b')
bars2 = plt.bar(labels, values,width=0.8, bottom=bottom,color='c')

5.饼图

饼图作为常用的图形，我们也会经常用到，这里我们采用常用函数fig, ax = plt.subplots()的方式画图，它会创建一个Figure和一个或多个Axes对象,方便画多个图，下面提供一个饼图的示例代码：

import matplotlib.pyplot as plt
# 数据
labels = ['Category A', 'Category B', 'Category C', 'Category D']
sizes = [15, 30, 45, 10] # 各部分所占比例（总和为100或任意数值）
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']
explode = (0, 0.1, 0, 0)  # 突出显示 'Category B'

fig, ax = plt.subplots(figsize=(6, 6)) # 设置图表大小，确保是正方形

ax.pie(sizes, explode=explode, labels=labels, colors=colors,
    autopct='%1.1f%%', shadow=True, startangle=140)
ax.axis('equal')  # 确保饼图是正圆，而不是椭圆
ax.set_title("Distribution of Categories")
plt.show()

运行这段代码，你会看到展示各类别比例的饼图(下图)，其中ax.pie()参数分别代表:占比、是否突出显示(“Category B”部分被轻微“分离”出来，以示强调)、名称、颜色、占比小数位、是否阴影以及角度。

6.散点图

散点图是通过一系列点的坐标位置来显示数据分布情况，是探索线性关系、聚类结构、离群点等的重要工具，也是常见的一种图形，在 Matplotlib 中，散点图由 ax.scatter() 或 plt.scatter() 生成，看例子：

import matplotlib.pyplot as plt
import numpy as np

np.random.seed(0)
x = np.random.randn(100)
y = np.random.randn(100)

fig, ax = plt.subplots()
sc = ax.scatter(x, y, alpha=0.7)
ax.set_title("Scatter Plot")
ax.set_xlabel("X")
ax.set_ylabel("Y")

plt.show()

运行上面代码，即可得到一张散点图， 上面的散点图是真的散，但是通常而言散点图用于寻找某种规律，因此可能会存在多类散点结合的情况，见下面的例子，通过构建两类散点分布，进行曲线拟合，寻找规律。

import numpy as np
import matplotlib.pyplot as plt

# 1. 生成示例数据：两类近似线性关系的散点
np.random.seed(0)
n1, n2 = 50, 50

# 第一类：y ≈ 2x + 1
x1 = np.linspace(0, 5, n1)
y1 = 2 * x1 + 1 + np.random.randn(n1) * 0.5      # 加一点噪声

# 第二类：y ≈ 2.2x + 1
x2 = np.linspace(0, 5, n2)
y2 = 2.2 * x2 + 1 + np.random.randn(n2) * 0.5   # 加一点噪声

# 2. 对两类数据分别做线性拟合：y = kx + b
k1, b1 = np.polyfit(x1, y1, 1)
k2, b2 = np.polyfit(x2, y2, 1)

# 为画直线生成平滑的 x
x_fit = np.linspace(0, 5, 100)
y1_fit = k1 * x_fit + b1
y2_fit = k2 * x_fit + b2

# 3. 作图
fig, ax = plt.subplots(figsize=(7, 5))

# 两类散点
ax.scatter(x1, y1, color="C0", marker="o", alpha=0.8, label="Class 1 data")
ax.scatter(x2, y2, color="C1", marker="s", alpha=0.8, label="Class 2 data")

# 对应拟合直线
ax.plot(x_fit, y1_fit, color="C0", linestyle="-", label=f"Class 1 fit: y = {k1:.2f}x + {b1:.2f}")
ax.plot(x_fit, y2_fit, color="C1", linestyle="--", label=f"Class 2 fit: y = {k2:.2f}x + {b2:.2f}")

# 4. 美化
ax.set_title("Scatter Plot with Two Linear Classes and Their Fits")
ax.set_xlabel("x")
ax.set_ylabel("y")
ax.legend()
ax.grid(True, linestyle="--", alpha=0.4)

plt.tight_layout()
plt.show()

运行上面的程序，即可得两种数据的分布，当然这里构建的数据偏理想，真实科研数据会有差异。

7.直方图

直方图用于显示数值型数据的分布情况。它将数据划分成一系列的“箱子”（bins），然后统计每个箱子中数据点的数量，绘制直方图主要用plt.hist()函数,照例先看代码：

import matplotlib.pyplot as plt
import numpy as np

# 生成随机的正态分布数据
np.random.seed(42)
data = np.random.randn(1000) # 生成 1000 个服从标准正态分布的随机数

fig, ax = plt.subplots(figsize=(6, 4))
hist_data = ax.hist(data, bins=30, color='skyblue', edgecolor='black', alpha=0.7, density=False)

ax.set_title("Histogram of Normally Distributed Data")
ax.set_xlabel("Value")
ax.set_ylabel("Frequency (Count)") # 如果 density=False，则为 Frequency
# ax.set_ylabel("Probability Density") # 如果 density=True，则为 Probability Density

# 可以在图上添加均值线（可选）
mean_value = np.mean(data)
ax.axvline(mean_value, color='red', linestyle='dashed', linewidth=1.5, label=f'Mean: {mean_value:.2f}')
ax.legend()

plt.show()

执行上述程序，得到如下的图，hist函数代码中data代表要绘制的数据，bins代表箱子的数量或边缘,可以是一个整数（如30），Matplotlib 会自动计算箱子边缘，也可以手动指定箱子边缘；color柱子的颜色；edgecolor柱子边框的颜色；alpha柱子的透明度；density如果为 True，则返回每个箱子的概率密度，而不是计数。此时，所有柱子面积之和为 1。

8.箱线图

箱线图（Boxplot）是一种用于表示分布统计特征的图形，它展示了中位数（median）、上四分位数 / 下四分位数（Q1, Q3）、四分位距（IQR = Q3 − Q1）、异常值（outliers）、最大值/最小值（非异常部分）等信息，对科研和数据分析尤其重要。 箱线图需要构建一组数据的list，程序中的data，每个元素需要有对应数据值，不能有空值，然后通过ax.boxplot()来画图,可以通过对应的参数来控制箱线图。继续看代码：

import matplotlib.pyplot as plt
import numpy as np

# 生成三组示例数据
np.random.seed(0)
data1 = np.random.normal(loc=0, scale=1, size=200)   # 组1
data2 = np.random.normal(loc=1, scale=0.8, size=200) # 组2
data3 = np.random.normal(loc=-0.5, scale=1.2, size=200) # 组3

# 数据以 list 的形式输入
data = [data1, data2, data3]

fig, ax = plt.subplots(figsize=(7, 5))

# 绘制箱线图
bp = ax.boxplot(
    data,
    notch=True,          # 是否绘制凹口（显示中位数置信区间）
    patch_artist=True,   # 允许填充颜色
    showfliers=True,     # 是否显示异常值
    widths=0.5           # 箱子的宽度
)

# 为不同箱子填充颜色（常用于科研图）
colors = ['#1f77b4', '#ff7f0e', '#2ca02c']
for patch, color in zip(bp['boxes'], colors):
    patch.set_facecolor(color) #设置填充颜色
    patch.set_alpha(0.4) #设置透明度

ax.set_title("Boxplot Example (Three Data Groups)")
ax.set_xticks([1, 2, 3])
ax.set_xticklabels(["Group 1", "Group 2", "Group 3"])
ax.set_ylabel("Value")

plt.show()
